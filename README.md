# 🎓 Student Marks Predictor

A Machine Learning web application built using **Python** and **Streamlit** that predicts a student's placement chances based on academic performance and provides interactive data analysis.

## 🚀 Features

- 📊 Interactive Streamlit Dashboard
- 📂 Upload and analyze student dataset
- 📈 Data visualization using Matplotlib
- 📋 Dataset preview and statistics
- 🤖 Student marks/placement prediction
- 💻 Simple and user-friendly interface

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib
- NumPy

## 📁 Project Structure

```
StudentMarksPrediction/
│── studentMarksPredictor.py
│── college_student_placement_dataset.csv
│── README.md
```

## ▶️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/StudentMarksPrediction.git
```

2. Move to the project folder

```bash
cd StudentMarksPrediction
```

3. Install the required libraries

```bash
pip install streamlit pandas matplotlib numpy
```

4. Run the application

```bash
streamlit run studentMarksPredictor.py
```

## 📸 Output

The application provides:

- Dataset Overview
- Data Analysis
- Graphical Visualizations
- Student Prediction Dashboard

The project uses the **College Student Placement Dataset** for analysis and prediction.
